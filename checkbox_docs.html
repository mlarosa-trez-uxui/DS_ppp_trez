Checkbox Button

A selection component that combines a circular icon with a text label. Allows the user to enable or disable an option within a form or list. Available in two icon positions, with or without helper text.

Do's & Don'ts
✓ DO
Use Leading when the icon is the primary action trigger.
Use Trailing when the label content is more important visually.
Always include Helper Text when the option might cause confusion.
Keep the label on a single line whenever possible.
Use Disabled state with a visible reason in the helper text.
✗ DON'T
Use this for exclusive selection — use Radio Button instead.
Truncate the label with ellipsis — reduce font size instead.
Change icon colors outside the system tokens.
Use Disabled without a clear reason for the user.
Mix Leading and Trailing in the same form group.
